HTML5学习指南

通过meta name=viewport...来让浏览器获取设备的实际宽度， 而不是屏幕分辨率

Width： viewport宽度，等于 device-width 为设备的宽度 （单位为缩放为 100% 时的 CSS 的像素）。 height：和 width 相对应，指定高度。 initial-scale：当页面第一次 load 时初始缩放比例。 maximum-scale：允许用户缩放到的最大比例。 minimum-scale：允许用户缩放到的最小比例。 user-scalable：用户是否可以手动缩放

border-radius属性定义四个角的弧度，四个值表示四个角
